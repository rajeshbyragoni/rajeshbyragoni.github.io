Angular Introduction Steps


Step 1 : Install the Angular CLI

npm install @angular/cli (latest cli)
npm install @angular/cli@6.0.2
npm install @angular/cli@6.2.3

Step 2 : Create a workspace and initial application

ng new projectname

Step 3: Run the application

ng serve
ng serve --host 0.0.0.0 --port 4201
ng serve --port 4201

Step 4: Install Packages in Angular

npm install --save @ng-bootstrap/ng-bootstrap
npm install jquery
npm install bootstrap@3.3.7
npm install font-awesome --save
npm install --save ng5-slider (price slider)
npm install ngx-chips --save (autocomplete)

** Step 5: Install app-routing in Angular5 only in angular6 it will come autometic

ng generate module app-routing --flat --module=app

lazy routing commans here

ng g module modulename --routing

inside module we have to create component so for that command
ng g c foldername/componentname --module=componentname
Ex :
ng g c login-pages/login --module=login

Step 5: Generate Service in Angular

ng generate service [servicename]
ng g service auth
ng g service --spec=false
ng g s services/common (with folder)

Step 6: Generate guard auth in Angular

ng generate guard auth

Step 7: Generate pipe in Angular

ng generate pipe [pipename]

Step 8: Generate ng build in Angular

ng build --prod (upload to server)
